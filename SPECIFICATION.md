# MathQuest — Техникалық Тапсырма

## Платформа туралы
Мұғалім ПК-да курс жасайды. Оқушы телефонда Duolingo сияқты өтеді.

## РӨЛДЕР
### Мұғалім
- Тіркелу / Кіру
- Курс жасау
- Курс ішіне траектория құру
- Траекторияға сабақ қосу
- Сабаққа материал қосу (видео, файл, мәтін, ойын)
- Оқушыларды курсқа қосу
- Прогресті көру

### Оқушы
- Кіру
- Маған тағайындалған курстарды көру
- Курсты кезекпен өту
- Сабақ ішінде материалды көру / ойын ойнау
- Прогресімді көру

## НЕГІЗГІ ЛОГИКА
### Курс құрылымы:
Курс └── Бөлім (Section) └── Сабақ (Lesson) └── Материалдар: ├── Мәтін ├── Видео ├── Файл (PDF) └── Ойын

### Өту ережесі:
- Алдыңғы сабақты бітірмей келесісі ашылмайды
- Сабақ бітті = барлық материалды көрді + ойынды өтті
- Ойынды өту = минимум 60% дұрыс жауап

## ОЙЫНДАР (5 дана)
1. **Жылдам есеп**: Сұрақ + 4 нұсқа + таймер
2. **Дұрыс/Бұрыс**: Тіркес шығады → 2 батырма
3. **Бомба**: 30 сек ішінде максимум жауап
4. **Санау тізбегі**: Бос орынға сан теру
5. **Рулетка**: Дөңгелек айналады → сұрақ шығады

## ПРОГРЕСС
- Әр сабақ: өтілді ✅ / өтілмеді 🔒
- Курс беті: "3/8 сабақ өтілді — 37%"
- Мұғалім көреді: кім қанша % өткен

## ЭКРАНДАР
### Мұғалім (ПК):
- Dashboard — курстар тізімі + жасау батырмасы
- Курс редактору — бөлімдер + сабақтар траекториясы
- Сабақ редактору — материал қосу (drag & drop)
- Оқушылар — тізім + прогресс

### Оқушы (Телефон/Планшет):
- Менің курстарым — карточкалар
- Курс жолы — Duolingo стиліндегі деңгейлер жолы
- Сабақ беті — материалдар кезекпен
- Ойын беті — толық экран

## ШЕКТЕУЛЕР
- Оқушы тек мұғалім қосқан курсты ғана көреді
- Мұғалім тек өз курстарын өңдей алады
- Ойынсыз сабақ болуы мүмкін
- Сабақсыз бөлім болмайды.
